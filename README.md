# MySQL Practice

SQL practice queries solved using MySQL Workbench, covering concepts from basics to intermediate level as part of my data analyst portfolio preparation.

## Topics Covered
- SELECT, WHERE, ORDER BY
- Aggregate functions (COUNT, SUM, AVG, MAX, MIN)
- GROUP BY & HAVING
- JOINs (INNER, LEFT, RIGHT, CROSS)
- Subqueries
- Window functions
- Normalization concepts (1NF–BCNF)

## Tools Used
- MySQL Workbench
- MySQL 8.0

## About
Each `.sql` file contains practice questions along with queries and comments explaining the logic. This repository is part of my ongoing SQL learning journey while preparing for Data Analyst roles.
